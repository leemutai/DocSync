Overview

Doctor's Appointment App is a full-stack web application that allows patients to book appointments with doctors. It includes user authentication, role-based access, and real-time updates. The system is designed for efficiency, scalability, and security.
🔥 Tech Stack
Frontend:

    Framework: React.js (Vite)

    Styling: Tailwind CSS

    State Management: Context API

    Authentication: JWT

    API Calls: Axios

Backend:

    Framework: Node.js with Express.js

    Authentication: JWT-based authentication

    API Documentation: Swagger

Database:

    DBMS: MongoDB

    ORM: Mongoose

Third-Party Services & APIs:

    Payment Gateway: M-Pesa API

    Notifications: Twilio (SMS) & Nodemailer (Emails)

Deployment & DevOps:

    Backend: Hosted on Render/Vercel

    Frontend: Deployed on Netlify/Vercel

    Database: MongoDB Atlas

    Containerization: Docker

    Version Control: Git & GitHub
